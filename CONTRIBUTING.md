# Contributing

Thanks for your interest in contributing!

## How to contribute

- Open an issue describing the bug/improvement before large changes.
- Fork the repo and create a branch from `main`.
- Submit small and focused PRs.

## Best practices

- Keep the existing code style.
- Avoid formatting-only changes when not necessary.
- Clearly describe what changed and why.
- If behavior changes, document it in the README.
